Check if this service is right for you

Test kit registration is how the NHS matches your contact details to the test kit you used at home or a test site.

Who should use this service

Use this service to register a coronavirus (COVID-19) test kit so you can get the test result if any of the following apply:

  • you were given a Test Registration Card at a test site
  • you're doing a rapid lateral flow test at a test site
  • you're taking part in surge testing and you're doing a PCR test at home or at a surge test site
  • you've entered the UK from another country, and you're doing a PCR test

Who should not use this service

You need to use a different service if:

  • you ordered a PCR test, and you need to register your PCR home test kit
  • you did a rapid lateral flow test yourself at home or work, and you need to report your test result to the NHS

What you need

Before you start, you will need:

  • the unique test kit barcode
  • a mobile phone number we can send the result to
  • the Royal Mail barcode - only if you are registering a PCR test that you will return by post
  • the NHS number for each person taking the test, if you have it - look up your NHS number

If you are registering a test for more than one person, you need to register each test to get all the results.

Get help to register your test kit

If you cannot use the online service, call 119 (free from mobiles and landlines). Lines are open every day, 7am to 11pm. 119 provides support in 200 languages.

InterpreterNow is a free online British Sign Language interpreter service for 119.

You can use the free Be My Eyes app to get help from trained NHS Test and Trace staff. Download the app, go to 'Specialized Help' and select NHS Test & Trace in the 'Personal Health' category.
