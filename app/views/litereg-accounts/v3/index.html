Use this service to register a coronavirus (COVID-19) test kit so you can get the test result if:

  • you were given a Test Registration Card at a test site
  • you're doing a rapid lateral flow test at a test site
  • you're taking part in surge testing and you're doing a PCR test
  • you've arrived in the UK after international travel and you're doing a PCR test

If you are registering a test for more than one person, you need to register each test to get all the results.

Before you start

You'll need:

  • the test kit barcode for the test you're registering
  • a mobile phone number (for you or the person you're registering it for)
  • the Royal Mail barcode - only if you are registering a PCR test that you will return by post
  • the NHS number for each person taking the test (called a CHI number in Scotland and H&C number in Northern Ireland)

The NHS number is optional but helps us quickly save your test result to your GP record.

In England, entering your NHS number also means your test results can be quickly saved to an ‘NHS COVID Pass’. You may be asked to show your pass to get into some events, or to travel abroad.

Get help to register your test kit

If you cannot use the online service, call 119 (free from mobiles and landlines). Lines are open every day, 7am to 11pm. You can get support in 200 languages.

InterpreterNow is a free online British Sign Language interpreter service for 119.

If you're blind or partially sighted, you can use the free Be My Eyes app to get help with registering. Download the app and go to the 'Specialist Help' section.
