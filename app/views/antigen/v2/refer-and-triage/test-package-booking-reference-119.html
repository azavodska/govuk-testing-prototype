Travel test reference number

Look for this in the order confirmation email sent by your travel test provider.

It's the same number you used on your Passenger Locator Form (PLF). It will also be in your PLF confirmation email.

It is 12 digits long for example, DANTH0037300.

If you cannot find the test reference number, contact the travel test provider. If the provider cannot give the reference number, call 119.
