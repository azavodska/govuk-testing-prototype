Use this service to register a (COVID-19) test for you so you can get the test result.

Use a different service if you need to report the result of a rapid lateral flow test you did at home or work.

You need to register each coronavirus test to get the results.

Before you start

You will need:

  • a mobile phone number we can send the result to
  • the test kit barcode - if you’re registering a PCR test
  • the test strip ID number - if you’re registering a lateral flow test
  • the Royal Mail barcode - if you’re registering a PCR swab test that you will return by post

If you are registering a test for more than one person, you need to register each test to get all the results.

Get help to register a test

If you cannot use the online service, call 119 (free from mobiles and landlines). Lines are open every day, 7am to 11pm. 119 provides support in 200 languages.

SignVideo is a free online British Sign Language interpreter service for 119.

You can use the free Be My Eyes app to get help from trained NHS Test and Trace staff. Download the app, go to ’Specialized Help’ and select NHS Test & Trace in the ‘Personal Health’ category.
