Use this service to get a collect code so you can pick up rapid lateral flow test packs from a pharmacy in England.

Your collect code does not reserve test packs for you, and tests may not be available at your preferred pharmacy.

We'll send your collect code by email or text. You do not need a collect code in Scotland, Wales and Northern Ireland. Find where to pick up a test near you on NHS.UK.

In England, most people without COVID-19 symptoms do not need to get lateral flow tests. Check who should get rapid lateral flow tests.

Start now

Collect your tests

Take your collect code to a pharmacy. You can use your code at any pharmacy that offers this service.

You can pick up rapid lateral flow tests for your household or another household. You should get a separate collect code for each household.

If you're aged 15 and under your parent or guardian should pick up the rapid lateral flow tests.

Someone will record your code and give you the rapid lateral flow tests. Collect codes help the NHS understand how rapid lateral flow tests are being used in England.

You should get a new collect code for each collection.
