Use this service to get a collect code so you can pick up rapid lateral flow test packs from:

  • a COVID-19 test site collection point
  • your school, college, university or workplace collection point

You do not need a collect code to get test packs from a pharmacy. Other ways to get a rapid lateral flow test.

If you’re 11 to 15, your parent or guardian must use this service to get a collect code. You can only pick up test packs from a school collection point.

How it works

1. Get a collect code

Before you go to the collection point, use this service to get a collect code. This is a code that you need to show when you collect your test packs. You can create an account so it’s quicker next time.

You can also get a collect code by calling 119 (free from landlines and mobiles). Lines are open from 7am to 11pm.

We’ll send your collect code by email or text. You can print your code.

2. Collect your tests

Take your collect code to the collection point. Someone will scan your code and give you the tests.

3. Do the test

After you’ve done the test, you should report your test result. We’ll tell you how to report your result when we send your collect code. You can also find this information in the test instructions.
